# Part Inventory

Demo project showning how to build REST API using Scala and Play framework


## Build and run

- SBT run
```
sbt compile 
```

```
sbt run 
```

- Bin run
```
sbt stage
```

```
./target/universal/stage/bin/partsinventory

```

Access: localhost:9000
